**타이틀: 어플리케이션 연동 기능 요건 초과**

**확인 내용** 
어플리케이션 연동 기능요건(5초 이내)

**재현 절차**
1. 상위 어플리케이션에서, 임의의 결제수단을 선택한다.
2. 결제 개시 버튼을 누른다.
3. 결제 어플리케이션 기동 ~ 카드 읽기 화면까지 5초 이내에 표시되는지 확인한다.

**기대 결과**
5초 이내에 카드 읽기 화면이 완전히 로드될 것.

**비고**
4회 실시하였으나 평균 6.4초가 걸림.

・시험 ID : TC-001
・어플리케이션 버전 : 1.2.0-release
・이용 단말 : MCP-1

**타이틀: 결제 중단 시 복귀 흐름 오류**

**확인 내용** 
결제 중단 시 복귀 flow

**재현 절차**
1. SUICA 결제를 선택 후 카드 읽기 화면까지 진행한다.
2. 카드 읽기 화면에서, 작업 전환 키를 누르고 어플리케이션을 종료한다.
3. 어플리케이션을 재기동 후, 이전의 결제가 정상적으로 종료되지 않았다는 메시지를 확인한다.
4. 3의 메시지에서 "예"를 선택 후 WAON 결제를 선택해 카드 읽기 화면까지 진행한다.
5. 결제 종류를 확인한다.

**기대 결과**
결제 종류가 WAON이 되어있다.

**비고**
3에서 메시지가 표시되지 않음. 5에서 결제 종류가 SUICA로 표시됨. 버그라고 생각합니다.

・시험 ID : TC-003
・어플리케이션 버전 : 1.2.0-release
・이용 단말 : MCP-1
